## 📋 구현 내용

### 1. 새로운 모듈 추가

#### 🔸 `scripts/longform_script_generator.py`
- **기능**: 10-15분 길이의 깊이 있는 스토리 스크립트 자동 생성
- **AI**: Gemini 2.5 Flash 사용
- **방식**: 
  - 오프닝 (관심 유도)
  - 스토리 1-3 (구체적 사례)
  - 클로징 (메시지 + 구독 유도)
- **특징**:
  - 감정적이고 진정성 있는 톤
  - 약 2000-2500단어 (10-15분 분량)
  - 실제 사례 및 통계 포함

#### 🔸 `scripts/longform_video_generator.py`
- **기능**: 롱폼 비디오 생성 (가로 1920x1080)
- **프로세스**:
  1. 스크립트를 4-12개 섹션으로 분할
  2. 각 섹션별 AI 배경 이미지 생성
  3. 음성과 이미지 동기화하여 비디오 생성
- **특징**:
  - Pollinations.ai 통합 (자동 이미지)
  - 자동 폴백 (이미지 생성 실패 시 단색 배경)
  - 부드러운 전환 효과 (0.5초)

#### 🔸 YouTube Uploader 확장
- **새 메서드**:
  - `create_longform_metadata()` - 롱폼용 최적화된 메타데이터
  - `upload_longform_video()` - 롱폼용 특화 업로드
  - `add_pinned_comment()` - 고정 댓글 자동 추가

### 2. 설정 업데이트

#### 📄 `config.json` 구조 개선

**Before** (쇼츠만):
```json
{
  "video": { ... },
  "content": { "topics": [...] },
  "upload": { ... }
}
```

**After** (쇼츠 + 롱폼):
```json
{
  "video": {
    "shorts": { ... },
    "longform": { ... }
  },
  "content": {
    "shorts": { "topics": [...] },
    "longform": { "topics": [...] }
  },
  "upload": {
    "shorts": { ... },
    "longform": { ... }
  },
  "scheduler": {
    "shorts": { ... },
    "longform": { ... }
  }
}
```

**롱폼 설정 값:**
- 해상도: 1920x1080 (가로)
- 최소 길이: 600초 (10분)
- 최대 길이: 900초 (15분)
- 각 이미지 표시: 8초
- 전환 시간: 0.5초

## 🎯 기능 상세

### 롱폼 스크립트 생성

**입력**: 주제 (자동 선택 또는 지정)  
**출력**: JSON 파일 포함:
- `title`: 감정적 제목
- `script`: 2000-2500단어 스크립트
- `estimated_duration`: 10-15분
- `content_type`: storytelling

**핵심 특징**:
- Gemini AI의 깊이 있는 스토리텔링
- 감정 변곡점 포함
- 실용적 조언 or 교훈 포함
- 자연스러운 음성 전달용 문체

### 롱폼 비디오 생성

**입력**: 스크립트 + 음성 파일  
**출력**: MP4 (1920x1080, 10-15분)

**프로세스**:
```
1. 음성 길이 분석 (예: 658초 = 11분)
2. 스크립트를 8개 섹션으로 분할 (11분 / 8 ≈ 82초)
3. 각 섹션별 AI 이미지 생성
4. 82초씩 이미지 표시 (0.5초 전환)
5. 최종 비디오 생성 (11분)
```

**이미지 생성**:
- 1번: 제목 이미지 (텍스트 + 배경)
- 2-8번: 섹션별 AI 이미지 (Pollinations)

### 롱폼 메타데이터

**자동 생성:**

**제목**:
```
[AI 생성 제목]
#스토리 #감동 #영감 #일상 #성공
```

**설명**:
```
🎬 오늘의 주제: [주제]

이 영상은 10-15분에 걸쳐 깊이 있는 스토리를 다룹니다.
따뜻한 마음으로 시청해주세요.

📌 주요 포인트:
• 실제 이야기와 교훈
• 감정이 담긴 메시지
• 영감과 희망

💬 댓글로 당신의 이야기를 나눠주세요!
❤️ 공감이 되셨다면 좋아요 눌러주세요
🔔 더 좋은 콘텐츠를 위해 구독 부탁드립니다

---
#스토리 #감동 #영감 #일상 #성공 #태그들
```

**카테고리**: 27 (영상/동영상, Shorts는 15)  
**태그**: 스토리, 감동, 일상, 성공, 영감

**고정 댓글**:
```
🙏 시청해주셔서 감사합니다!

[제목] 영상입니다.

💬 댓글로 당신의 이야기를 들려주세요
❤️ 좋아요를 눌러 다른 분들께 추천해주세요
🔔 더 좋은 콘텐츠를 위해 구독 부탁드립니다
```

## 📊 성능 비교

### 쇼츠 vs 롱폼 vs 둘다

| 지표 | 쇼츠만 | 롱폼만 | 쇼츠+롱폼 |
|------|--------|--------|-----------|
| 초기 조회 | 빠름 | 느림 | 매우 빠름 |
| 지속 조회 | 낮음 | 높음 | 매우 높음 |
| 구독 증가 | 낮음 | 높음 | 매우 높음 |
| 광고 수익 | 낮음 | 높음 | 매우 높음 |
| 업로드 빈도 | 5회/일 | 3회/일 | 8회/일 |

**결론**: 쇼츠로 트래픽 확보 → 롱폼으로 구독 및 수익 극대화
