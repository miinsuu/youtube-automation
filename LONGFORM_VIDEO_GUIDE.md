# 🎬 롱폼 비디오 (10-15분) 자동화 가이드

## 📋 개요

YouTube 채널의 조회수와 구독자를 확보하기 위해 **롱폼 비디오(10-15분)** 기능을 추가했습니다.

### 특징
- ✅ **깊이 있는 스토리텔링**: 감정적인 내용 전달
- ✅ **자동 이미지 생성**: AI 배경 이미지 포함
- ✅ **최적화된 메타데이터**: YouTube 롱폼용 제목, 설명, 해시태그
- ✅ **고정 댓글**: 자동 추가되는 채널 소개 댓글
- ✅ **정기 스케줄링**: 매일 자동 생성 및 업로드

---

## 🎯 사용 방법

### 1. 로컬 테스트 (롱폼 비디오 생성)

```bash
# 롱폼 비디오 1개 생성 (업로드 제외)
python main.py --type longform --no-upload

# 특정 주제로 롱폼 비디오 생성
python main.py --type longform --topic "성공한 사람들의 일상 습관"

# 업로드 포함
python main.py --type longform
```

### 2. 쇼츠 + 롱폼 동시 생성

```bash
# 쇼츠 1개 + 롱폼 1개 동시 생성
python main.py --type both
```

### 3. 수동 GitHub Actions 실행

**GitHub 저장소 → Actions 탭 → "YouTube Shorts Auto Upload" → "Run workflow"**

옵션:
- **Video type**: `shorts` / `longform` / `both`
- **Number of videos**: 1-3 (쇼츠만 해당)
- **Enable YouTube upload**: `true` / `false`

## 🎨 롱폼 비디오 생성 과정

```
1️⃣ 스크립트 생성
   → Gemini AI가 깊이 있는 스토리 작성
   → 약 2000-2500단어 (10-15분 분량)

2️⃣ 음성 생성
   → Edge TTS (ko-KR-SunHiNeural, 1.2x 속도)
   → 약 10-15분 길이의 MP3 생성

3️⃣ 이미지 생성
   → 각 섹션별 AI 배경 이미지 (Pollinations.ai)
   → 총 4-12개 이미지

4️⃣ 비디오 합성
   → MoviePy를 사용해 이미지 + 음성 결합
   → 가로 해상도 (1920x1080)
   → 자막 미포함 (음성만 사용)

5️⃣ YouTube 업로드
   → 최적화된 메타데이터 자동 적용
   → 고정 댓글 자동 추가
```

## 📊 영상 길이 기준

### 롱폼 비디오 (권장: 10-15분)

**이유:**
- 유튜브 알고리즘: 10분 이상 동영상에서 광고 수익 2배
- 시청 유지: 깊이 있는 콘텐츠로 시청 시간 증가
- 구독 확보: 스토리텔링으로 감정적 연결 형성
- SEO 효과: 긴 영상이 검색 순위에 유리

**파마미터 설정** (config.json):
```json
"longform": {
  "resolution": "1920x1080",
  "min_duration_seconds": 600,      // 최소 10분
  "max_duration_seconds": 900,      // 최대 15분
  "image_display_seconds": 8,       // 각 이미지 표시 시간
  "transition_seconds": 0.5         // 이미지 전환 시간
}
```


### 이미지 생성 실패

**증상**: "🎨 AI 이미지 생성 실패"

**대체 방법**:
- config.json에서 `use_ai_background: false`로 설정
- 단색 배경으로 자동 생성됨

```json
"longform": {
  "use_ai_background": false
}
```
